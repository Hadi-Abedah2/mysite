Welcome to my Django project!

This is index page, and I'll be progressively adding more apps and features as part of my ongoing practice.

In the 'Ads' app, you can:
  • Perform basic CRUD operations after registering.
  • register users using conventional or social login (using google account)
  • Mark ads as favorites or remove them from favorites.
  • Search for specific ads.
  • Access related ads by clicking on their tags.
good but not exhaustive test cases are written for this app.
In the 'Sentiment Analyzer' app, you can:
  • write in texts and let the AI decides if it is:
    • Positive
    • Negative
    • Neutral
the app is tested thorughly and have rate limit of 8000 requests/month.
in the 'Courses' app, you can:
  • enroll in a course
  • browse lessons(currently no further lesson details will be available)
  • take unlimited tests and get your score with a feedback.
this app features need more testing currently.
In the 'APIs' app, you can:
  • test the API endpoints designed to expose the funcionality of the 'Ads' app.
  • register and authenticate clients using session based authentication or token based authentication(JWT Tokens).
  • make CRUD operations on ads and comments(appropriate permissions were implemented).
  • search for or filter ads based on their titles or price ranges

API root endpoints should be self explainatory

use known Djoser endpoints for user management found at https://djoser.readthedocs.io/en/latest/jwt_endpoints.html

add query parameters to implement the filtering and searching capabilities on ads "?price_gte={integer}, price_lte={integer}, ?title={string}"
